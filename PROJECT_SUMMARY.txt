HOW TO USE:
================================================================================

1. Install: pip install -r requirements.txt
2. Configure: Create .env with EMAIL_USER and EMAIL_PASS
3. Run: streamlit run streamlit.py
4. Upload contacts.csv
5. Write your email subject and body
6. Use {name} for personalization
7. Set delay and send

That's it!
